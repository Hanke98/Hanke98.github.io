Node.js 初探

发表于 2017-11-11 | 分类于 Node.js 开发笔记

我的博客

这一周，为了完成我的网络课程设计，我熬了两个晚上，制作了一个简单的基于Node.js 和 MongoDB的博客，算是比年初自己写的哪一个稍微进步了一点，下面来简单介绍一下我的这个博客的搭建过程。

首先来看一下我的博客首页

起步

我的博客使用的是Express框架
首先我们来创建一个Express 应用

express -e blog --view=hbs

这里我们选择了Handlebars作为我们的模版引擎，Handlebars是全球使用率最高的模板引擎,所以当之无愧是全球最受欢迎的模板引擎。之前我使用的是jade模版，对比了jade和ejs还有这次使用的Handlebars，我发现jade语法太过奇怪，不像是我们常见的开发用语，ejs有几乎是纯粹的html的语法又不支持继承。所以最后还是选择了Handlebars。

前端页面

页面使用了Boostrap框架。深感Boostrap的强大，以后还是要深入的学习一下。同时在生成前端页面的时候，我使用了Handlebars，用来实现组件循环渲染

这里 each是Handlebars定义好的 helpler 就像是我们在JSP中的c: foreach，后端渲染时只需将post变量设置好即可。

数据库

这一次采用了MongoDB, MongoDB 是一个从 NoSQL 的数据库，数据以json文档的形式来进行存储，这样的方式让开发人员不必掌握复杂的SQL语句，就可以轻松的实现数据库操作。在Express中我们通常使用的是Mongoose来驱动我们的数据库
